# stic-server

# Install

```bash
$ npm install -g stic-server
```

## Usage

```bash
# Start serving using port 3000 and current directory ./
$ stic-server

# Serve in different port
$ stic-server --port 5000

# Serve directory
$ stic-server --dir "./public"

# Launch the default browser
$ stic-server --open

# Disable server log
$ stic-server --no-log

# Help
$ stic-server --help
```
